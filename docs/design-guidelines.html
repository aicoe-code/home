AICoE Design Guidelines

Internal reference document - Not linked from public pages

Color Palette

Primary Background

Gradient: #160c26 to #06040c at 135deg

Deep purple gradient providing a sophisticated, modern foundation

Accent Colors

Mint Green (Primary Accent): #a8e6cf

Used for: Taglines, subtle highlights, secondary text, floating orbs

Vibrant Pink (Secondary Accent): #ff6b9d

Used for: Key statements, important emphasis, call-to-action highlights

Text Colors

  • Primary Text: #ffffff (white)
  • Secondary Text: #999999 (gray - for taglines)

Typography

Font Family

'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

Font Sizes & Weights

  • H1 (Main Title): 4rem, weight 300, letter-spacing -0.02em
  • Tagline: 1.5rem, weight 300
  • Mission Statement (Large): 3rem, weight 300, line-height 1.6
  • Mission Statement (Small): 2rem, weight 300, line-height 1.6
  • Links: 1.1rem
  • Section Titles: 1.2rem, uppercase, letter-spacing 0.3em
Note: Use font-weight 300 for minimal, elegant feel. Use weight 400 for emphasis within statements.

Layout & Spacing

Container

  • Max-width: 800px (home) / 900px (mission)
  • Centered with flexbox
  • Min-height: 100vh
  • Padding: 20px

Vertical Spacing

  • Between title and tagline: 1rem
  • Between tagline and links: 4rem
  • Between section title and content: 4rem
  • Between emphasized statements: 2rem

Interactive Elements

Links

  • Color: #ffffff
  • No underline by default
  • Border-bottom: 1px solid transparent
  • Transition: all 0.3s ease

Link Hover State

  • Border-bottom-color: #ffffff
  • Transform: scale(1.05)

Link Active/Click State

  • Transform: scale(1.15)

Back Link

  • Color: #a8e6cf
  • Hover: translateX(-5px)

Decorative Elements

Floating Orbs

  • Count: 3 orbs per page
  • Sizes: 200px, 180px, 150px
  • Color: Radial gradient with mint green rgba(168, 230, 207, 0.4) to rgba(168, 230, 207, 0.05)
  • Filter: blur(40px)
  • Animation: Float animation with 20s, 25s, 30s durations
  • Position: Absolute positioning with varied placements

Float Animation

Smooth floating motion with translate and scale transformations:

  • 0%: translate(0, 0) scale(1)
  • 25%: translate(100px, -50px) scale(1.1)
  • 50%: translate(50px, 100px) scale(0.9)
  • 75%: translate(-80px, 50px) scale(1.05)
  • 100%: return to start

Responsive Design

Mobile Breakpoint: 768px

  • H1: Reduce to 2.5rem
  • Tagline: Reduce to 1.2rem
  • Mission Statement: Reduce to 2rem
  • Links: Stack vertically (flex-direction: column) with 1rem gap
  • Section Titles: Reduce letter-spacing to 0.2em

Design Principles

  • Minimal & Modern: Clean lines, ample white space, subtle animations
  • Elegant Typography: Large, thin fonts with negative letter-spacing
  • Centered Layouts: Content centered both horizontally and vertically
  • Subtle Interactions: Smooth transitions, gentle scaling effects
  • Atmospheric Background: Gradient with floating orbs for depth
  • Strategic Color Use: Sparingly use accent colors for maximum impact

Page Structure

Standard Page Template

  1. Background gradient
  2. Floating orbs (3)
  3. Container (centered, z-index: 1)
  4. Optional back link
  5. Content section
Consistency: All pages should maintain the same background, orb styling, and core typography for brand cohesion.
